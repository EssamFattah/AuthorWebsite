A RESTful HTTP API

This web API provides access to a server that is used to store photos and data submitted by the user. It covers the CRUD functionality and defines the REST elements of each service, including the endpoint URI, HTTP methods, and content-type of the response.

CRUD Functionality

Create

  • Endpoint URI: '/add'
  • HTTP method: 'POST'
  • Content-Type: 'json'
  • Input: JSON data sent in the body of the HTTP request

    • Example Input:

    { "author": "testname",

    "alt": "Image of testname",

    "tags": "tag,author,science",

    "image": "https://example.jpg",

    "image": "https://example.jpg",

    "description": "The internet and the Web aren't the same thing." }


  • Output: status code 500 (server problem that prevents it from fulfilling request) in case of error, 200 in case of success

Read

  • Endpoint URI: '/{id}'
  • HTTP method: 'GET'
  • Content-Type: 'json'
  • Input: None
  • Output: status code 404 (server cant find the request rescource) or 500 (server problem that prevents it from fulfilling request) in case of error, 200 in case of successful request

Update

  • Endpoint URI: '/items/{id}'
  • HTTP method: 'PUT'
  • Content-Type: 'json'
  • Input: JSON data sent in the body of the HTTP request

    • Example Input:

    { "author": "testname",

    "alt": "Image of testname",

    "tags": "tag,author,science",

    "image": "https://example.jpg",

    "image": "https://example.jpg",

    "description": "The internet and the Web aren't the same thing." }


  • Output: status codes 400 (invalid request), 404 (server cant find the request rescource), 500 (server problem that prevents it from fulfilling request) in case of error, 204 (request has succeeded, but that the client doesn't need to navigate away from its current page.) in case of successful request

Delete

  • Endpoint URI: '/delete/{id}'
  • HTTP method: 'DELETE'
  • Content-Type: 'json'
  • Input: None
  • Output: status code 500 (server problem that prevents it from fulfilling request) in case of error, 200 in case of successful request
